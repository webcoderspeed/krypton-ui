# Alert Dialog

A modal dialog component that displays important information and requires user confirmation before proceeding. Built on top of Radix UI with full accessibility support and smooth animations.

## Installation

```bash
npm install krypton-ui
```

## Import

```tsx
import {
  AlertDialog,
  AlertDialogAction,
  AlertDialogCancel,
  AlertDialogContent,
  AlertDialogDescription,
  AlertDialogFooter,
  AlertDialogHeader,
  AlertDialogTitle,
  AlertDialogTrigger,
} from 'krypton-ui'
```

## Basic Usage

The AlertDialog component provides a way to display critical information that requires user confirmation.

## API Reference

### AlertDialog

The root component that manages the dialog state.

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| defaultOpen | boolean | false | The open state of the dialog when it is initially rendered |
| open | boolean | - | The controlled open state of the dialog |
| onOpenChange | function | - | Event handler called when the open state changes |

### AlertDialogTrigger

The button that opens the dialog.

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| asChild | boolean | false | Change the default rendered element for the one passed as a child |

### AlertDialogContent

The content area of the dialog.

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| className | string | - | Additional CSS classes |
| onEscapeKeyDown | function | - | Event handler called when the escape key is down |
| onPointerDownOutside | function | - | Event handler called when a pointer event occurs outside the bounds |

### AlertDialogHeader

Container for the dialog title and description.

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| className | string | - | Additional CSS classes |

### AlertDialogFooter

Container for the dialog action buttons.

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| className | string | - | Additional CSS classes |

### AlertDialogTitle

The title of the dialog.

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| className | string | - | Additional CSS classes |

### AlertDialogDescription

The description text of the dialog.

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| className | string | - | Additional CSS classes |

### AlertDialogAction

The primary action button that closes the dialog.

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| className | string | - | Additional CSS classes |
| asChild | boolean | false | Change the default rendered element for the one passed as a child |

### AlertDialogCancel

The cancel button that closes the dialog without performing the action.

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| className | string | - | Additional CSS classes |
| asChild | boolean | false | Change the default rendered element for the one passed as a child |

## Accessibility

- Supports keyboard navigation
- Focus is automatically trapped within the dialog
- Pressing `Esc` closes the dialog
- Clicking outside the dialog closes it
- Supports screen readers with proper ARIA attributes
- Focus returns to the trigger element when closed
