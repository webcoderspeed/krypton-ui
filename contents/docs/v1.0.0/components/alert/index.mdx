# Alert

A versatile alert component for displaying important messages, notifications, and contextual information to users. Supports multiple variants and can include icons, titles, and descriptions.

## Installation

```bash
npm install krypton-ui
```

## Import

```tsx
import { Alert, AlertDescription, AlertTitle } from 'krypton-ui'
```

## Basic Usage

The Alert component provides a simple way to display important information to users.

<CodePreview
  code={`import { Alert, AlertDescription, AlertTitle } from 'krypton-ui'

export default function BasicAlert() {
  return (
    <Alert>
      <AlertTitle>Heads up!</AlertTitle>
      <AlertDescription>
        You can add components and dependencies to your app using the cli.
      </AlertDescription>
    </Alert>
  )
}`}
  preview={
    <Alert>
      <AlertTitle>Heads up!</AlertTitle>
      <AlertDescription>
        You can add components and dependencies to your app using the cli.
      </AlertDescription>
    </Alert>
  }
/>

### Default Alert

The default variant provides a neutral appearance for general information.

<CodePreview
  code={`import { Alert, AlertDescription, AlertTitle } from 'krypton-ui'

  preview={
    <Alert>
      <AlertTitle>Information</AlertTitle>

### Destructive Alert

Use the destructive variant for error messages and critical warnings.

<CodePreview
  code={`import { Alert, AlertDescription, AlertTitle } from 'krypton-ui'

    </Alert>

  preview={
    <Alert variant="destructive">

## API Reference

### Alert

The main alert container component.

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| variant | "default" \| "destructive" | "default" | The visual style variant of the alert |
| className | string | - | Additional CSS classes |
| children | ReactNode | - | The content of the alert |

### AlertTitle

The title component for the alert.

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| className | string | - | Additional CSS classes |
| children | ReactNode | - | The title content |

### AlertDescription

The description component for the alert.

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| className | string | - | Additional CSS classes |
| children | ReactNode | - | The description content |

## Accessibility

- Uses the `alert` role for screen readers
- Supports keyboard navigation
- Proper color contrast for all variants
- Icons are properly labeled for screen readers
- Text content is readable and well-structured

## Best Practices

- Use the appropriate variant for the message type
- Include clear, actionable text
- Add relevant icons to improve visual communication
- Keep messages concise and easy to understand
- Use destructive variant sparingly for critical errors only
