# Aspect Ratio

A component that maintains a consistent aspect ratio for its content, ensuring proper proportions across different screen sizes and devices. Built on top of Radix UI AspectRatio primitive.

## Installation

```bash
npm install krypton-ui
```

## Import

```tsx
import { AspectRatio } from 'krypton-ui'
```

## Basic Usage

The AspectRatio component wraps content to maintain a specific width-to-height ratio.

<CodePreview
  code={`import { AspectRatio } from 'krypton-ui'

export default function BasicAspectRatio() {
  return (
    <div className="w-[450px]">
      <AspectRatio ratio={16 / 9}>
        <img
          src="https://images.unsplash.com/photo-1588345921523-c2dcdb7f1dcd?w=800&dpr=2&q=80"
          alt="Photo by Drew Beamer"
          className="rounded-md object-cover w-full h-full"
        />
      </AspectRatio>
    </div>
  )
}`}
  preview={

## API Reference

### AspectRatio

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| ratio | number | 1 | The desired aspect ratio (width / height) |
| className | string | - | Additional CSS classes |
| children | ReactNode | - | The content to be displayed within the aspect ratio container |

## Common Aspect Ratios

| Ratio | Value | Use Case |
| --- | --- | --- |
| 16:9 | 16/9 ≈ 1.78 | Widescreen videos, modern displays |
| 4:3 | 4/3 ≈ 1.33 | Traditional TV, computer monitors |
| 3:2 | 3/2 = 1.5 | Photography, print media |
| 1:1 | 1 | Square images, social media posts |
| 21:9 | 21/9 ≈ 2.33 | Ultra-wide displays, cinematic content |
| 9:16 | 9/16 = 0.56 | Mobile screens, vertical videos |
| Golden Ratio | 1.618 | Aesthetically pleasing proportions |

## Accessibility

- Maintains proper proportions across different screen sizes
- Works with screen readers and assistive technologies
- Supports keyboard navigation for interactive content
- Preserves image alt text and other accessibility attributes

## Best Practices

- Use consistent aspect ratios throughout your application
- Choose appropriate ratios for your content type
- Consider responsive design when selecting aspect ratios
- Test on different screen sizes to ensure proper display
- Use object-cover for images to maintain aspect ratio without distortion
