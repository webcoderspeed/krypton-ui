## API Reference

### AspectRatio

Built on top of [Radix Aspect Ratio](https://www.radix-ui.com/primitives/docs/components/aspect-ratio).

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `ratio` | `number` | `1` | The desired ratio |
| `asChild` | `boolean` | `false` | Change the default rendered element for the one passed as a child |

## Styling

The AspectRatio component can be styled using Tailwind CSS classes:

```tsx
<AspectRatio ratio={16 / 9} className="bg-muted rounded-lg overflow-hidden">
  <img src="..." alt="..." className="object-cover w-full h-full" />
</AspectRatio>
```

## Accessibility

The AspectRatio component maintains the aspect ratio of its content while being fully accessible. When used with images, ensure proper `alt` attributes are provided.

## Common Ratios

- **16:9** - Standard widescreen (1.78)
- **4:3** - Traditional TV/monitor (1.33)
- **1:1** - Square (1)
- **3:2** - Classic photography (1.5)
- **21:9** - Ultra-wide (2.33)

## Best Practices

- Use appropriate ratios for your content type
- Ensure images have proper `object-fit` classes
- Consider responsive behavior with container sizing
- Provide meaningful alt text for images
- Test with various content sizes and types
