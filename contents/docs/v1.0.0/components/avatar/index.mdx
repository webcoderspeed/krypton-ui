## Import

<CodePreview code="import { Avatar, AvatarFallback, AvatarImage } from '@/components/ui/avatar'" language="tsx" />

## Usage

### Default

A basic avatar with an image and fallback text.

<CodePreview 
  code={`import { Avatar, AvatarFallback, AvatarImage } from "@/components/ui/avatar"

export default function AvatarDefault() {
  return (
    <Avatar>
      <AvatarImage src="https://github.com/shadcn.png" alt="@shadcn" />
      <AvatarFallback>CN</AvatarFallback>
    </Avatar>
  )
}`} 
  language="tsx" 
  preview={<AvatarDefault />} 
/>

### Fallback

Avatars with various fallback scenarios when images fail to load.

<CodePreview 
  code={`import { Avatar, AvatarFallback, AvatarImage } from "@/components/ui/avatar"

export default function AvatarFallback() {
  return (
    <div className="flex items-center space-x-4">
      <Avatar>
        <AvatarImage src="https://github.com/shadcn.png" alt="@shadcn" />
        <AvatarFallback>CN</AvatarFallback>
      </Avatar>
      <Avatar>
        <AvatarImage src="/broken-image.jpg" alt="Broken" />
        <AvatarFallback>BK</AvatarFallback>
      </Avatar>
      <Avatar>
        <AvatarFallback>JD</AvatarFallback>
      </Avatar>
      <Avatar>
        <AvatarFallback className="bg-blue-500 text-white">AB</AvatarFallback>
