## API Reference

### Breadcrumb

Built on top of semantic navigation elements for accessibility.

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `className` | `string` | - | Additional CSS classes to apply to the breadcrumb container. |

### BreadcrumbList

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `className` | `string` | - | Additional CSS classes to apply to the breadcrumb list. |

### BreadcrumbItem

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `className` | `string` | - | Additional CSS classes to apply to the breadcrumb item. |

### BreadcrumbLink

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `href` | `string` | - | The URL to navigate to when the link is clicked. |
| `className` | `string` | - | Additional CSS classes to apply to the breadcrumb link. |

### BreadcrumbPage

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `className` | `string` | - | Additional CSS classes to apply to the current page indicator. |

### BreadcrumbSeparator

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `className` | `string` | - | Additional CSS classes to apply to the separator. |

### BreadcrumbEllipsis

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `className` | `string` | - | Additional CSS classes to apply to the ellipsis indicator. |
