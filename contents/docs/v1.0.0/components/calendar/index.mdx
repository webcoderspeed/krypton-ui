### Disabled Dates

Disable specific dates or date ranges to prevent selection.

<CodePreview 
  code={`import { Calendar } from '@/components/ui/calendar'
import { useState } from 'react'

export default function CalendarDisabled() {
  const [date, setDate] = useState<Date | undefined>()

  return (
    <Calendar
      mode="single"
      selected={date}
      onSelect={setDate}
      disabled={(date) =>
        date > new Date() || date < new Date('1900-01-01')
      }
      initialFocus
      className="rounded-md border"
    />
  )
}`}

## API Reference

### Props

The Calendar component accepts all react-day-picker props plus:

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `mode` | `"single" | "multiple" | "range"` | `"single"` | The selection mode of the calendar |
| `selected` | `Date | Date[] | DateRange` | - | The selected date(s) |
| `onSelect` | `function` | - | Callback fired when date selection changes |
| `disabled` | `boolean | function | Date | Date[]` | `false` | Dates that should be disabled |
| `numberOfMonths` | `number` | `1` | Number of months to display |
| `defaultMonth` | `Date` | - | The month to display initially |
| `className` | `string` | - | Additional CSS classes to apply |
| `initialFocus` | `boolean` | `false` | Whether to focus the calendar on mount |

### Keyboard Navigation

- **Arrow Keys**: Navigate between dates
- **Enter** or **Space**: Select the focused date
- **Page Up/Down**: Navigate between months
- **Home/End**: Navigate to start/end of week
- **Tab**: Move focus to next focusable element
