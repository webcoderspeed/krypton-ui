# Carousel

A flexible and accessible carousel component that supports horizontal and vertical orientations, keyboard navigation, and programmatic control. Built on top of Embla Carousel with full TypeScript support.

## Installation

```bash
npm install krypton-ui
```

## Import

```tsx
import {
  Carousel,
  CarouselContent,
  CarouselItem,
  CarouselNext,
  CarouselPrevious,
} from 'krypton-ui'
```

## Basic Usage

The Carousel component provides a smooth sliding interface for displaying multiple items.

<CodePreview
  code={`import {
  Carousel,
  CarouselContent,
  CarouselItem,
  CarouselNext,
  CarouselPrevious,
} from 'krypton-ui'
import { Card, CardContent } from 'krypton-ui'

export default function BasicCarousel() {
  return (
    <Carousel className="w-full max-w-xs">
      <CarouselContent>
        {Array.from({ length: 5 }).map((_, index) => (
          <CarouselItem key={index}>
            <div className="p-1">
              <Card>
                <CardContent className="flex aspect-square items-center justify-center p-6">
                  <span className="text-4xl font-semibold">{index + 1}</span>
                </CardContent>
              </Card>
            </div>
          </CarouselItem>
        ))}
      </CarouselContent>
      <CarouselPrevious />
      <CarouselNext />
    </Carousel>
  )
}`}

## API Control

Use the carousel API for programmatic control and state management.

<CodePreview
  code={`import {
  Carousel,
  CarouselContent,
  CarouselItem,
  CarouselNext,
  CarouselPrevious,
  type CarouselApi,
} from 'krypton-ui'
import { Card, CardContent } from 'krypton-ui'
import { Button } from 'krypton-ui'
import React, { useState, useEffect } from 'react'

export default function CarouselApiExample() {
  const [api, setApi] = useState<CarouselApi>()
  const [current, setCurrent] = useState(0)
  const [count, setCount] = useState(0)

  useEffect(() => {
    if (!api) {
      return
    }

    setCount(api.scrollSnapList().length)
    setCurrent(api.selectedScrollSnap() + 1)

    api.on("select", () => {
      setCurrent(api.selectedScrollSnap() + 1)
    })
  }, [api])

  return (
    <div className="mx-auto max-w-xs">
      <Carousel setApi={setApi} className="w-full max-w-xs">
        <CarouselContent>
          {Array.from({ length: 5 }).map((_, index) => (
            <CarouselItem key={index}>
              <Card>
                <CardContent className="flex aspect-square items-center justify-center p-6">
                  <span className="text-4xl font-semibold">{index + 1}</span>
                </CardContent>
              </Card>
            </CarouselItem>
          ))}
        </CarouselContent>
        <CarouselPrevious />
        <CarouselNext />
      </Carousel>
      <div className="py-2 text-center text-sm text-muted-foreground">
        Slide {current} of {count}
      </div>
      <div className="flex justify-center gap-2 mt-4">
        <Button
          variant="outline"
          size="sm"
          onClick={() => api?.scrollTo(0)}
        >
          Go to first
        </Button>
        <Button
          variant="outline"
          size="sm"
          onClick={() => api?.scrollTo(count - 1)}
        >
          Go to last
        </Button>
      </div>
    </div>
  )
}`}
  preview={<CarouselApiExample />}
/>

## Props

### Carousel

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| orientation | string | horizontal | The orientation of the carousel |
| opts | object | undefined | Embla Carousel options |
| plugins | array | undefined | Embla Carousel plugins |
| setApi | function | undefined | Callback to get the carousel API |

### CarouselContent

Inherits all props from HTMLDivElement.

### CarouselItem

Inherits all props from HTMLDivElement.

### CarouselPrevious / CarouselNext

Inherits all props from the Button component.

## Keyboard Navigation

The carousel supports keyboard navigation:

- **Arrow Left**: Navigate to previous slide (horizontal)
- **Arrow Right**: Navigate to next slide (horizontal)
- **Arrow Up**: Navigate to previous slide (vertical)
- **Arrow Down**: Navigate to next slide (vertical)

## Accessibility

The carousel component follows WAI-ARIA guidelines:

- Uses proper ARIA roles and labels
- Supports keyboard navigation
- Provides screen reader announcements
- Maintains focus management
