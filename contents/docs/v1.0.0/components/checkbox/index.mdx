## API Reference

### Props

The Checkbox component accepts all Radix UI Checkbox props plus:

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `checked` | `boolean | "indeterminate"` | - | The controlled checked state of the checkbox |
| `defaultChecked` | `boolean` | `false` | The default checked state (uncontrolled) |
| `onCheckedChange` | `(checked: boolean | "indeterminate") => void` | - | Callback fired when the checked state changes |
| `disabled` | `boolean` | `false` | Whether the checkbox is disabled |
| `required` | `boolean` | `false` | Whether the checkbox is required |
| `name` | `string` | - | The name attribute for form submission |
| `value` | `string` | `"on"` | The value attribute for form submission |
| `id` | `string` | - | The id attribute for accessibility |
| `className` | `string` | - | Additional CSS classes to apply |

### States

- **Unchecked**: Default state, checkbox is not selected
- **Checked**: Checkbox is selected
- **Indeterminate**: Partially checked state (useful for parent checkboxes)
- **Disabled**: Checkbox cannot be interacted with

### Keyboard Navigation

- **Space**: Toggles the checkbox state
- **Tab**: Moves focus to the next focusable element
- **Shift + Tab**: Moves focus to the previous focusable element

## Accessibility

The Checkbox component follows WAI-ARIA guidelines:

- Uses semantic checkbox input with proper ARIA attributes
- Supports keyboard navigation
- Provides proper focus management
- Associates labels correctly with `htmlFor` and `id`
- Announces state changes to screen readers
- Maintains proper color contrast ratios

## Best Practices

1. **Always provide labels**: Use clear, descriptive labels for checkboxes
2. **Group related checkboxes**: Use fieldsets for related checkbox groups
3. **Consider indeterminate state**: Use for parent checkboxes controlling child checkboxes
4. **Provide clear feedback**: Show validation states and error messages
5. **Use consistent spacing**: Maintain consistent spacing between checkboxes and labels
