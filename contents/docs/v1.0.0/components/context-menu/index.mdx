---
title: Context Menu
description: Displays a menu to the user — such as a set of actions or functions — triggered by a right click.
---

## API Reference

### ContextMenu

The root component that wraps the trigger and content.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | The trigger and content components |

### ContextMenuTrigger

The element that triggers the context menu on right-click.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `asChild` | `boolean` | `false` | Change the default rendered element |
| `children` | `ReactNode` | - | The trigger content |
| `className` | `string` | - | Additional CSS classes |

### ContextMenuContent

The component that pops out when the context menu is open.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `sideOffset` | `number` | `4` | The distance in pixels from the trigger |
| `children` | `ReactNode` | - | The menu items |

### ContextMenuItem

The component that contains the context menu items.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `asChild` | `boolean` | `false` | Change the default rendered element |
| `className` | `string` | - | Additional CSS classes |
| `inset` | `boolean` | `false` | Add left padding for alignment |
| `disabled` | `boolean` | `false` | Whether the item is disabled |
| `onSelect` | `(event: Event) => void` | - | Event handler called when the user selects an item |
| `children` | `ReactNode` | - | The item content |

### ContextMenuCheckboxItem

An item that can be controlled and rendered like a checkbox.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `checked` | `boolean` | - | The controlled checked state |
| `onCheckedChange` | `(checked: boolean) => void` | - | Event handler called when the checked state changes |
| `className` | `string` | - | Additional CSS classes |
| `disabled` | `boolean` | `false` | Whether the item is disabled |
| `children` | `ReactNode` | - | The item content |

### ContextMenuRadioItem

An item that can be controlled and rendered like a radio button.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | The unique value of the item |
| `className` | `string` | - | Additional CSS classes |
| `disabled` | `boolean` | `false` | Whether the item is disabled |
| `children` | `ReactNode` | - | The item content |

### ContextMenuRadioGroup

Used to group multiple `ContextMenuRadioItem`s.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | The controlled value of the radio group |
| `onValueChange` | `(value: string) => void` | - | Event handler called when the value changes |
| `children` | `ReactNode` | - | The radio items |

### ContextMenuLabel

Used to render a label. It won't be focusable using arrow keys.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `inset` | `boolean` | `false` | Add left padding for alignment |
| `children` | `ReactNode` | - | The label content |

### ContextMenuSeparator

Used to visually separate items in the context menu.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |

### ContextMenuShortcut

A small utility to help display keyboard shortcuts.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | The shortcut text |

### ContextMenuSub

Contains all the parts of a submenu.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | The sub trigger and content |

### ContextMenuSubTrigger

An item that opens a submenu.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `inset` | `boolean` | `false` | Add left padding for alignment |
| `disabled` | `boolean` | `false` | Whether the item is disabled |
| `children` | `ReactNode` | - | The trigger content |

### ContextMenuSubContent

The component that pops out when a submenu is open.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | The submenu items |
