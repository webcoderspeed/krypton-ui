## API Reference

### ContextMenu

The root container for the context menu.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | The trigger and content components |
| `onOpenChange` | `(open: boolean) => void` | - | Callback when the menu open state changes |
| `modal` | `boolean` | `true` | Whether the menu should be modal |

### ContextMenuTrigger

The element that triggers the context menu on right click.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `asChild` | `boolean` | `false` | Render as child element |
| `children` | `ReactNode` | - | The trigger element |
| `disabled` | `boolean` | `false` | Whether the trigger is disabled |

### ContextMenuContent

The container for the menu items.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | The menu items |
| `sideOffset` | `number` | `4` | Offset from the trigger |
| `alignOffset` | `number` | `0` | Alignment offset |
| `avoidCollisions` | `boolean` | `true` | Whether to avoid collisions |

### ContextMenuItem

A menu item that can be activated.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | The item content |
| `disabled` | `boolean` | `false` | Whether the item is disabled |
| `onSelect` | `(event: Event) => void` | - | Callback when item is selected |
| `inset` | `boolean` | `false` | Whether to add left padding |

### ContextMenuCheckboxItem

A menu item with a checkbox.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | The item content |
| `checked` | `boolean \| 'indeterminate'` | `false` | The checked state |
| `onCheckedChange` | `(checked: boolean) => void` | - | Callback when checked state changes |
| `disabled` | `boolean` | `false` | Whether the item is disabled |

### ContextMenuRadioItem

A menu item with a radio button.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | The item content |
| `value` | `string` | - | The value of the radio item |
| `disabled` | `boolean` | `false` | Whether the item is disabled |

### ContextMenuRadioGroup

A group of radio items.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | The selected value |
| `onValueChange` | `(value: string) => void` | - | Callback when value changes |
| `children` | `ReactNode` | - | The radio items |

### ContextMenuLabel

A label for grouping menu items.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | The label content |
| `inset` | `boolean` | `false` | Whether to add left padding |

### ContextMenuSeparator

A visual separator between menu items.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |

### ContextMenuShortcut

Displays keyboard shortcuts in menu items.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | The shortcut text |

### ContextMenuSub

Container for submenu functionality.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | SubTrigger and SubContent |
| `onOpenChange` | `(open: boolean) => void` | - | Callback when submenu open state changes |

### ContextMenuSubTrigger

Trigger for opening a submenu.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | The trigger content |
| `inset` | `boolean` | `false` | Whether to add left padding |
| `disabled` | `boolean` | `false` | Whether the trigger is disabled |

### ContextMenuSubContent

Container for submenu items.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | The submenu items |
| `sideOffset` | `number` | `2` | Offset from the trigger |
| `alignOffset` | `number` | `-5` | Alignment offset |

## Keyboard Navigation

The Context Menu component supports full keyboard navigation:

- **Right Click / Menu Key**: Open context menu
- **Escape**: Close menu
- **Arrow Keys**: Navigate between items
- **Enter / Space**: Activate item
- **Tab**: Navigate to next focusable element
- **Shift + Tab**: Navigate to previous focusable element
- **Arrow Right**: Open submenu (when available)
- **Arrow Left**: Close submenu

## Accessibility

The Context Menu component follows WAI-ARIA guidelines:

- Uses proper ARIA roles and attributes
- Supports keyboard navigation
- Manages focus correctly
- Provides screen reader announcements
- Supports high contrast mode
- Respects reduced motion preferences

### ARIA Attributes

- `role="menu"` on menu content
- `role="menuitem"` on menu items
- `role="menuitemcheckbox"` on checkbox items
- `role="menuitemradio"` on radio items
- `aria-expanded` on submenu triggers
- `aria-checked` on checkbox and radio items
- `aria-disabled` on disabled items

## Best Practices

1. **Use meaningful labels**: Provide clear, descriptive text for menu items
2. **Group related actions**: Use separators and labels to organize menu items
3. **Include keyboard shortcuts**: Show shortcuts for frequently used actions
4. **Limit menu depth**: Avoid deeply nested submenus for better usability
5. **Use icons consistently**: Include icons for visual clarity when appropriate
6. **Handle destructive actions**: Use visual cues (like red text) for dangerous actions
7. **Provide feedback**: Show loading states or confirmations for actions
8. **Consider mobile**: Context menus may not work well on touch devices
