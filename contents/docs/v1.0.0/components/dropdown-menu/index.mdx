# Dropdown Menu

Displays a menu to the user — such as a set of actions or functions — triggered by a button. Perfect for navigation menus, action menus, or any contextual options.

## Installation

```bash
npm install @radix-ui/react-dropdown-menu
npm install lucide-react
```

## Import

```tsx
import {
  DropdownMenu,
  DropdownMenuContent,
  DropdownMenuItem,
  DropdownMenuCheckboxItem,
  DropdownMenuRadioItem,
  DropdownMenuLabel,
  DropdownMenuSeparator,
  DropdownMenuShortcut,
  DropdownMenuGroup,
  DropdownMenuPortal,
  DropdownMenuSub,
  DropdownMenuSubContent,
  DropdownMenuSubTrigger,
  DropdownMenuRadioGroup,
  DropdownMenuTrigger,
} from "@/components/ui/dropdown-menu"
```

## API Reference

### DropdownMenu

The root container for the dropdown menu.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `open` | `boolean` | - | The controlled open state |
| `defaultOpen` | `boolean` | `false` | The default open state (uncontrolled) |
| `onOpenChange` | `(open: boolean) => void` | - | Callback when open state changes |
| `modal` | `boolean` | `true` | Whether the dropdown is modal |
| `dir` | `"ltr" \| "rtl"` | - | Reading direction |
| `children` | `ReactNode` | - | The dropdown content |

### DropdownMenuTrigger

The button that opens the dropdown menu.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `asChild` | `boolean` | `false` | Render as child component |
| `children` | `ReactNode` | - | The trigger content |

### DropdownMenuContent

The dropdown menu content container.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `align` | `"start" \| "center" \| "end"` | `"center"` | Alignment relative to trigger |
| `alignOffset` | `number` | `0` | Offset from the align axis |
| `side` | `"top" \| "right" \| "bottom" \| "left"` | `"bottom"` | Preferred side |
| `sideOffset` | `number` | `4` | Distance from the trigger |
| `collisionPadding` | `number \| Partial<Record<Side, number>>` | `0` | Padding for collision detection |
| `avoidCollisions` | `boolean` | `true` | Whether to avoid collisions |
| `forceMount` | `boolean` | `false` | Force mount the content |
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | The content |

**Default Styles:**
- `z-50 min-w-[8rem] overflow-hidden` - Z-index and layout
- `rounded-md border bg-popover p-1` - Appearance
- `text-popover-foreground shadow-md` - Typography and shadow
- `data-[state=open]:animate-in` - Opening animation
- `data-[state=closed]:animate-out` - Closing animation

### DropdownMenuItem

A menu item within the dropdown.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `disabled` | `boolean` | `false` | Whether the item is disabled |
| `onSelect` | `(event: Event) => void` | - | Callback when item is selected |
| `textValue` | `string` | - | Text value for typeahead |
| `asChild` | `boolean` | `false` | Render as child component |
| `className` | `string` | - | Additional CSS classes |
| `inset` | `boolean` | `false` | Add left padding for alignment |
| `children` | `ReactNode` | - | The item content |

**Default Styles:**
- `relative flex cursor-default select-none` - Layout and interaction
- `items-center rounded-sm px-2 py-1.5` - Spacing and shape
- `text-sm outline-none transition-colors` - Typography and transitions
- `focus:bg-accent focus:text-accent-foreground` - Focus states
- `data-[disabled]:pointer-events-none data-[disabled]:opacity-50` - Disabled state

### DropdownMenuCheckboxItem

A checkbox menu item.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `checked` | `boolean \| "indeterminate"` | - | The checked state |
| `onCheckedChange` | `(checked: boolean) => void` | - | Callback when checked state changes |
| `disabled` | `boolean` | `false` | Whether the item is disabled |
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | The item content |

### DropdownMenuRadioItem

A radio menu item.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | The radio value |
| `disabled` | `boolean` | `false` | Whether the item is disabled |
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | The item content |

### DropdownMenuRadioGroup

Groups radio items together.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | The selected value |
| `onValueChange` | `(value: string) => void` | - | Callback when value changes |
| `children` | `ReactNode` | - | The radio items |

### DropdownMenuLabel

A label for grouping menu items.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `inset` | `boolean` | `false` | Add left padding for alignment |
| `children` | `ReactNode` | - | The label content |

**Default Styles:**
- `px-2 py-1.5 text-sm font-semibold` - Spacing and typography

### DropdownMenuSeparator

A visual separator between menu items.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |

**Default Styles:**
- `-mx-1 my-1 h-px bg-muted` - Spacing and appearance

### DropdownMenuShortcut

Displays keyboard shortcuts.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | The shortcut text |

**Default Styles:**
- `ml-auto text-xs tracking-widest opacity-60` - Typography and positioning

### DropdownMenuGroup

Groups related menu items.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | The grouped items |

### DropdownMenuSub

A submenu container.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `open` | `boolean` | - | The controlled open state |
| `defaultOpen` | `boolean` | `false` | The default open state |
| `onOpenChange` | `(open: boolean) => void` | - | Callback when open state changes |
| `children` | `ReactNode` | - | The submenu content |

### DropdownMenuSubTrigger

The trigger for a submenu.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `disabled` | `boolean` | `false` | Whether the trigger is disabled |
| `className` | `string` | - | Additional CSS classes |
| `inset` | `boolean` | `false` | Add left padding for alignment |
| `children` | `ReactNode` | - | The trigger content |

### DropdownMenuSubContent

The content of a submenu.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | The submenu items |

### DropdownMenuPortal

Portals submenu content to a different DOM location.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `forceMount` | `boolean` | `false` | Force mount the portal |
| `container` | `HTMLElement` | - | Portal container |
| `children` | `ReactNode` | - | The portal content |

### Language Selector

```tsx
function LanguageSelector() {
  const [language, setLanguage] = useState('en')
  
  const languages = [
    { value: 'en', label: 'English' },
    { value: 'es', label: 'Español' },
    { value: 'fr', label: 'Français' },
    { value: 'de', label: 'Deutsch' },
  ]
  
  return (
    <DropdownMenu>
      <DropdownMenuTrigger asChild>
        <Button variant="outline">
          {languages.find(lang => lang.value === language)?.label}
        </Button>
      </DropdownMenuTrigger>
      <DropdownMenuContent>
        <DropdownMenuRadioGroup value={language} onValueChange={setLanguage}>
          {languages.map((lang) => (
            <DropdownMenuRadioItem key={lang.value} value={lang.value}>
              {lang.label}
            </DropdownMenuRadioItem>
          ))}
        </DropdownMenuRadioGroup>
      </DropdownMenuContent>
    </DropdownMenu>
  )
}
```

## Best Practices

1. **Clear Labels**: Use descriptive labels for menu items
2. **Logical Grouping**: Group related items with separators
3. **Keyboard Shortcuts**: Include shortcuts for common actions
4. **Icons**: Use consistent icons to improve recognition
5. **Disabled States**: Clearly indicate disabled items
6. **Destructive Actions**: Use warning colors for destructive actions
7. **Mobile Friendly**: Ensure touch targets are large enough
8. **Loading States**: Handle async actions gracefully
9. **Consistent Positioning**: Use consistent alignment across your app
10. **Accessibility**: Test with screen readers and keyboard navigation

## Troubleshooting

### Menu Not Opening

Ensure the trigger is properly configured:

```tsx
<DropdownMenuTrigger asChild>
  <Button>Open Menu</Button>
</DropdownMenuTrigger>
```

### Positioning Issues

Adjust the positioning props:

```tsx
<DropdownMenuContent
  align="end"
  side="bottom"
  sideOffset={4}
  alignOffset={0}
>
```

### Submenu Not Working

Ensure you're using the portal:

```tsx
<DropdownMenuSub>
  <DropdownMenuSubTrigger>Submenu</DropdownMenuSubTrigger>
  <DropdownMenuPortal>
    <DropdownMenuSubContent>
      {/* submenu items */}
    </DropdownMenuSubContent>
  </DropdownMenuPortal>
</DropdownMenuSub>
```

### Styling Issues

Ensure Tailwind CSS is properly configured and the component styles are included in your build.
