## Installation

```bash
npx shadcn-ui@latest add label
```

## Usage

```tsx
import { Label } from "@/components/ui/label"
```

```tsx
<Label htmlFor="email">Your email address</Label>
```

## Manual Installation

1. Install the following dependency:

```bash
npm install @radix-ui/react-label
```

2. Copy and paste the following code into your project.

```tsx
"use client"

import * as React from "react"
import * as LabelPrimitive from "@radix-ui/react-label"
import { cva, type VariantProps } from "class-variance-authority"

import { cn } from "@/lib/utils"

const labelVariants = cva(
  "text-sm font-medium leading-none peer-disabled:cursor-not-allowed peer-disabled:opacity-70"
)

const Label = React.forwardRef<
  React.ElementRef<typeof LabelPrimitive.Root>,
  React.ComponentPropsWithoutRef<typeof LabelPrimitive.Root> &
    VariantProps<typeof labelVariants>
>(({ className, ...props }, ref) => (
  <LabelPrimitive.Root
    ref={ref}
    className={cn(labelVariants(), className)}
    {...props}
  />
))
Label.displayName = LabelPrimitive.Root.displayName

export { Label }
```

3. Update the import paths to match your project setup.

## API Reference

### Label

Built on top of [Radix Label](https://www.radix-ui.com/docs/primitives/components/label).

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `htmlFor` | `string` | - | The id of the element the label is associated with |
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | The label content |

## Accessibility

The Label component is built on top of Radix UI's Label primitive, which provides:

- **Proper association**: Uses the `htmlFor` attribute to associate with form controls
- **Screen reader support**: Announces the label when the associated control receives focus
- **Click handling**: Clicking the label focuses the associated control
- **Disabled state**: Automatically handles disabled styling when the associated control is disabled

### Best Practices

1. **Always associate labels**: Use the `htmlFor` attribute to associate labels with form controls
2. **Required indicators**: Use visual indicators (like asterisks) for required fields
3. **Error states**: Apply appropriate styling and colors for validation errors
4. **Descriptive text**: Provide clear, concise label text that describes the expected input
5. **Additional context**: Use `aria-describedby` for additional help text or instructions

### Example with Accessibility Features

```tsx
<div className="space-y-2">
  <Label htmlFor="username">
    Username <span className="text-red-500">*</span>
  </Label>
  <Input
    id="username"
    type="text"
    aria-describedby="username-help"
    required
  />
  <p id="username-help" className="text-sm text-muted-foreground">
    Choose a unique username between 3-20 characters.
  </p>
</div>
```

## Styling

The Label component uses a simple variant system with the following default styles:

- `text-sm`: Small text size
- `font-medium`: Medium font weight
- `leading-none`: Tight line height
- `peer-disabled:cursor-not-allowed`: Disabled cursor when associated control is disabled
- `peer-disabled:opacity-70`: Reduced opacity when associated control is disabled

### Custom Styling

You can customize the label appearance by passing additional classes:

```tsx
<Label htmlFor="field" className="text-lg text-primary">
  Custom Styled Label
</Label>
```
