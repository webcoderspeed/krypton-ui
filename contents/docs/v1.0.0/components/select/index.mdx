# Select

Displays a list of options for the user to pick fromâ€”triggered by a button. Perfect for forms, filters, and any interface where users need to choose from a predefined set of options.

## Installation

```bash
npm install @radix-ui/react-select
```

## Import

```tsx
import {
  Select,
  SelectContent,
  SelectItem,
  SelectTrigger,
  SelectValue,
} from "@/components/ui/select"
```

### Multi-Select Alternative

```tsx
import { useState } from "react"
import { Badge } from "@/components/ui/badge"
import { Button } from "@/components/ui/button"
import {
  Select,
  SelectContent,
  SelectItem,
  SelectTrigger,
  SelectValue,
} from "@/components/ui/select"
import { X } from "lucide-react"

const skills = [
  "JavaScript", "TypeScript", "React", "Vue", "Angular",
  "Node.js", "Python", "Java", "C#", "Go", "Rust", "PHP"
]

export function MultiSelectAlternative() {
  const [selectedSkills, setSelectedSkills] = useState<string[]>([])
  const [currentValue, setCurrentValue] = useState("")

  const addSkill = (skill: string) => {
    if (!selectedSkills.includes(skill)) {
      setSelectedSkills([...selectedSkills, skill])
    }
    setCurrentValue("")
  }

  const removeSkill = (skill: string) => {
    setSelectedSkills(selectedSkills.filter(s => s !== skill))
  }

  const availableSkills = skills.filter(skill => !selectedSkills.includes(skill))

  return (
    <div className="space-y-4 max-w-md">
      <div>
        <label className="text-sm font-medium">Skills</label>
        <div className="mt-1">
          <Select value={currentValue} onValueChange={addSkill}>
            <SelectTrigger>
              <SelectValue placeholder="Add a skill" />
            </SelectTrigger>
            <SelectContent>
              {availableSkills.map((skill) => (
                <SelectItem key={skill} value={skill}>
                  {skill}
                </SelectItem>
              ))}
            </SelectContent>
          </Select>
        </div>
      </div>

      {selectedSkills.length > 0 && (
        <div className="space-y-2">
          <label className="text-sm font-medium">Selected Skills:</label>
          <div className="flex flex-wrap gap-2">
            {selectedSkills.map((skill) => (
              <Badge key={skill} variant="secondary" className="pr-1">
                {skill}
                <Button
                  variant="ghost"
                  size="sm"
                  className="ml-1 h-auto p-0 text-muted-foreground hover:text-foreground"
                  onClick={() => removeSkill(skill)}
                >
                  <X className="h-3 w-3" />
                </Button>
              </Badge>
            ))}
          </div>
        </div>
      )}
    </div>
  )
}
```

### Searchable Select

```tsx
import { useState } from "react"
import {
  Select,
  SelectContent,
  SelectItem,
  SelectTrigger,
  SelectValue,
} from "@/components/ui/select"
import { Input } from "@/components/ui/input"

const cities = [
  "New York", "Los Angeles", "Chicago", "Houston", "Phoenix",
  "Philadelphia", "San Antonio", "San Diego", "Dallas", "San Jose",
  "Austin", "Jacksonville", "Fort Worth", "Columbus", "Charlotte",
  "San Francisco", "Indianapolis", "Seattle", "Denver", "Washington"
]

export function SearchableSelect() {
  const [searchTerm, setSearchTerm] = useState("")
  const [selectedCity, setSelectedCity] = useState("")

  const filteredCities = cities.filter(city =>
    city.toLowerCase().includes(searchTerm.toLowerCase())
  )

  return (
    <div className="space-y-2">
      <Select value={selectedCity} onValueChange={setSelectedCity}>
        <SelectTrigger className="w-[250px]">
          <SelectValue placeholder="Select a city" />
        </SelectTrigger>
        <SelectContent>
          <div className="p-2">
            <Input
              placeholder="Search cities..."
              value={searchTerm}
              onChange={(e) => setSearchTerm(e.target.value)}
              className="h-8"
            />
          </div>
          {filteredCities.length === 0 ? (
            <div className="p-2 text-sm text-muted-foreground">
              No cities found.
            </div>
          ) : (
            filteredCities.map((city) => (
              <SelectItem key={city} value={city}>
                {city}
              </SelectItem>
            ))
          )}
        </SelectContent>
      </Select>
      {selectedCity && (
        <p className="text-sm text-muted-foreground">
          Selected: {selectedCity}
        </p>
      )}
    </div>
  )
}
```

## API Reference

### Select

The root select component.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | The controlled value |
| `defaultValue` | `string` | - | The default value (uncontrolled) |
| `onValueChange` | `(value: string) => void` | - | Callback when value changes |
| `open` | `boolean` | - | The controlled open state |
| `defaultOpen` | `boolean` | `false` | The default open state |
| `onOpenChange` | `(open: boolean) => void` | - | Callback when open state changes |
| `disabled` | `boolean` | `false` | Whether the select is disabled |
| `required` | `boolean` | `false` | Whether the select is required |
| `name` | `string` | - | The name attribute |
| `dir` | `"ltr" \| "rtl"` | - | Reading direction |
| `children` | `ReactNode` | - | The select content |

### SelectTrigger

The button that opens the select.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | The trigger content |
| `asChild` | `boolean` | `false` | Render as child component |

**Default Styles:**
- `flex h-10 w-full items-center justify-between` - Layout
- `rounded-md border border-input bg-background` - Appearance
- `px-3 py-2 text-sm ring-offset-background` - Spacing and typography
- `placeholder:text-muted-foreground` - Placeholder styling
- `focus:outline-none focus:ring-2 focus:ring-ring` - Focus states
- `disabled:cursor-not-allowed disabled:opacity-50` - Disabled state

### SelectValue

Displays the selected value or placeholder.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `placeholder` | `string` | - | Placeholder text |
| `className` | `string` | - | Additional CSS classes |

### SelectContent

The dropdown content container.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `position` | `"item-aligned" \| "popper"` | `"item-aligned"` | Positioning strategy |
| `side` | `"top" \| "right" \| "bottom" \| "left"` | `"bottom"` | Preferred side |
| `sideOffset` | `number` | `4` | Distance from trigger |
| `align` | `"start" \| "center" \| "end"` | `"center"` | Alignment relative to trigger |
| `alignOffset` | `number` | `0` | Offset from align axis |
| `avoidCollisions` | `boolean` | `true` | Whether to avoid collisions |
| `collisionBoundary` | `Element \| Element[]` | - | Collision boundary |
| `collisionPadding` | `number \| Partial<Record<Side, number>>` | `0` | Collision padding |
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | The content |

**Default Styles:**
- `relative z-50 max-h-96 min-w-[8rem]` - Layout and constraints
- `overflow-hidden rounded-md border bg-popover` - Appearance
- `text-popover-foreground shadow-md` - Typography and shadow
- `data-[state=open]:animate-in` - Opening animation
- `data-[state=closed]:animate-out` - Closing animation

### SelectItem

A selectable item within the dropdown.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | The item value (required) |
| `disabled` | `boolean` | `false` | Whether the item is disabled |
| `textValue` | `string` | - | Text value for typeahead |
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | The item content |

**Default Styles:**
- `relative flex w-full cursor-default select-none` - Layout and interaction
- `items-center rounded-sm py-1.5 pl-8 pr-2` - Spacing
- `text-sm outline-none` - Typography
- `focus:bg-accent focus:text-accent-foreground` - Focus states
- `data-[disabled]:pointer-events-none data-[disabled]:opacity-50` - Disabled state

### SelectLabel

A label for grouping select items.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | The label content |

**Default Styles:**
- `py-1.5 pl-8 pr-2 text-sm font-semibold` - Spacing and typography

### SelectSeparator

A visual separator between select items.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional CSS classes |

**Default Styles:**
- `-mx-1 my-1 h-px bg-muted` - Spacing and appearance

### SelectGroup

Groups related select items.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | The grouped items |

## Best Practices

1. **Clear Labels**: Use descriptive labels and placeholders
2. **Logical Grouping**: Group related options together
3. **Reasonable Options**: Limit the number of options (consider search for large lists)
4. **Default Values**: Provide sensible default selections when appropriate
5. **Validation**: Provide clear validation feedback
6. **Loading States**: Handle loading states for dynamic options
7. **Empty States**: Show appropriate messages for empty option lists
8. **Mobile Optimization**: Ensure touch targets are large enough
9. **Keyboard Support**: Test keyboard navigation thoroughly
10. **Consistent Styling**: Maintain consistent appearance across your app

## Troubleshooting

### Select Not Opening

Ensure the trigger is properly configured:

```tsx
<SelectTrigger>
  <SelectValue placeholder="Select..." />
</SelectTrigger>
```

### Value Not Updating

For controlled selects, ensure you're handling the change:

```tsx
const [value, setValue] = useState("")

<Select value={value} onValueChange={setValue}>
  {/* content */}
</Select>
```

### Positioning Issues

Adjust the positioning props:

```tsx
<SelectContent
  position="popper"
  side="bottom"
  align="start"
  sideOffset={4}
>
  {/* items */}
</SelectContent>
```

### Styling Issues

Ensure Tailwind CSS is properly configured and component styles are included in your build.

### Performance with Large Lists

For large option lists, consider implementing virtualization or search functionality to improve performance.
